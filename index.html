Web Development

w3schools

This site is a really good reference site for beginners. It has information on HTML, CSS and Javascript as well as some common web-development libraries. I don't know much about their tutorial but it exists

Free Code Camp

This site has several amazing tutorials. They will walk you through beginning front-end web-development with HTML, CSS and Javascript all the way through back-end development and app development. They have also added some new courses on data visualizaton and APIs. They're a pretty great resource and they give certificates if that motivates you.

Game Development Tutorials

Game From Scratch

This site has a whole host of game development tutorials. I haven't tried them out myself but they feature tutorials on several popular technologies including game engines such as Godot 3, Armory and LibGDX, Blender and other rendering software and several programming languages including Lua and C++. I recommend that you check it out and see if there's a tutorial for you.
